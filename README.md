# MIED 挑战性任务

一个纯静态网页互动小游戏，用转盘随机抽取社恐挑战任务。

## 功能

- 固定 12 格挑战转盘
- 从完整挑战库随机换一组 12 个任务
- 点击开始或再转一次时，只从当前转盘任务中抽取
- 箭头固定在 12 点方向，先确定中奖索引再反推旋转角度
- 抽中后高亮中奖格，并展示任务名称与具体行动
- 浅色多巴胺视觉风格，适配手机和电脑

## 文件

- `index.html`：页面结构
- `styles.css`：视觉样式
- `script.js`：挑战库和转盘交互逻辑

## 本地运行

这是纯静态网页，不需要后端。可以直接在浏览器打开 `index.html`。
